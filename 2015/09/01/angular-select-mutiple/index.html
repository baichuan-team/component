angular-select-mutiple

2015-09-01

支持单选, 多选, 框选组件, 按住shift框选为取消选择


已选元素个数:
                   

如何使用

安装

引用

Html

父元素增加属性select-multiple-box, 能被选中的子元素增加属性select-multiple-item

Css

.selected为选中以后的样式, .selecting为框选时, 所选中的元素的样式, 均可自定义

Js

引入selectMultiple即可

Callback

如需在选中后, 添加回调方法, 加上select-done=”func()”即可.

取消框选

如果页面中处理dom的事件较多, 框选会存在一定的性能问题, 加上smultiple-select=”false”即可关闭框选功能(默认为开启)
